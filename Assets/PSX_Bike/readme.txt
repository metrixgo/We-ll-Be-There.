- Unity -

Up above you have everything needed to import the model into unity.
If you aren't using the haunted ps1 render pipeline, you need to change the material shader in engine.

- Blender -

Just open the .blend file and import the texture and you're good to go.

- Other Engines -

Export the model to .FBX in blender and drag the model in your desired game engine or project.
